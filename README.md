# 템하키트 (Temhakit)

**템하키트**는 [템플릿하우스(Templatehouse)](https://kit.temha.io)에서 제공하는 CSS 프레임워크이자 퍼블리싱 키트입니다. 일관된 스타일 가이드와 컴포넌트·템플릿 예시를 한곳에서 보고, 필요한 코드를 복사해 프로젝트에 맞게 확장할 수 있도록 구성되어 있습니다.

## 이 저장소에서 할 수 있는 것

- **컴포넌트** — 버튼, 폼, 카드, 모달 등 UI 블록 HTML·스타일 참고
- **템플릿** — 샘플 페이지 구조와 리소스(css, js, 아이콘 등) 패턴 확인
- **리소스** — `templatehouse.css`, 공통 스크립트 등 배포용 에셋 활용
